Resizing Image Content In Proportion

.container {
width: 50%;
height: 200px;
overflow: hidden;
}

.container img {
max-width: 100%;
height: auto;
display: block;
}

Use the CSS above as it is a very common design pattern used to scale images and videos proportionally.
